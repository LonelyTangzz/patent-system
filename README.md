# 基于SpringBoot的专利系统网站
前端使用的Bootstrap4.1框架，引用JQuery地址、日期插件，数据交互图表使用的Echart，后端基于SpringBoot搭建的简单MVC框架.
功能：专利增加，查询，修改，删除；用户管理、分类管理、消息管理。
预期（后期将Mysql存储替换为hadoop分布式存储）
